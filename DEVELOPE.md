# Developer guidance

## Installation

Please follow these instruction to install rag-colls in dev mode.

- **Clone the repository**

```bash
git clone https://github.com/hienhayho/rag-colls.git
cd rag-colls/
```

- **Install uv:**

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

- **Install packages:**

```bash
uv pip install -v -e .
uv pip install pytest
```
